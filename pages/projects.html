RISC-V Emulator

An emulator for a 32 bit instruction set based off of the RV32I ISA. The emulator is written in C, and is capable of running 33 instructions. It supports programs written in C and compiled using the RISC-V GNU Compiler Toolchain as long as they only use the 33 instructions supported by the emulator.
